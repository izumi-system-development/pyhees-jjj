# .kiro フォルダについて

## このフォルダの目的

ドキュメンティングファイルの置場として .kiro フォルダを使用しています。
.kiro フォルダは AIツールKiro.Dev によって管理されるフォルダです。
何らかの理由で Kiro の使用が終了した時点で他の Documents 用フォルダを作成して移動します。

## Kiro.Dev とは

Kiro.DevはSpecification Driven Development（SDD：仕様駆動開発）を支援するAI開発アシスタントです。

SDDでは明確な仕様書を基にして開発を進めるため、ドキュメントの管理と維持が非常に重要になります。Kiroは以下の理由でドキュメント管理に適しています：

- **仕様書ベースの開発**: 仕様書から自動的にコードを生成・更新
- **ドキュメントとコードの同期**: 仕様変更時にコードとドキュメントを一貫して更新  
- **複雑なシステムの可視化**: 図解やフローチャートで理解しやすく表現
- **チーム協働の支援**: 技術者・非技術者問わず理解できる形でのドキュメント作成

Kiroは`.kiro`フォルダ内の仕様書や設定を参照してプロジェクトの文脈を理解するため、ここにドキュメントを配置することで効果的な開発支援を受けることができます。
